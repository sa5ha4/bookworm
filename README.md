apraksts:
Programma tiek palaista. Parādās izvēlne ar piecām izvēles iespējām:
Pievienot grāmatu (add book) - var pievienot grāmatu. Ja viss ir veiksmīgi, iespējams atgriezties izvēlnē vai pievienot vēl vienu grāmatu. Ja kaut kas neizdodas, var mēģināt pievienot grāmatu vēlreiz vai atgriezties izvēlnē.
Apskatīt visus (open the list) - tiek atvērts grāmatu saraksts. Var nospiest uz grāmatas un atzīmēt to kā izlasītu/neizlasītu, pēc tam aizvērt grāmatu un pāriet uz izvēlni. Var arī sakārtot sarakstu pēc hronoloģijas (pirms vai pēc 2000. gada) un pāriet uz izvēlni.
Dzēst (remove book) - var izdzēst jebkuru grāmatu no bibliotēkas. Ja viss izdodas, jūs atgriežaties izvēlnē, ja ne, varat mēģināt vēlreiz vai pāriet uz izvēlni.
Atrast (find book) - var atrast grāmatu pēc nosaukuma vai autora. Ja viss izdodas, jūs atgriežaties izvēlnē vai varat vēlreiz meklēt grāmatu. Ja neizdodas, varat mēģināt vēlreiz vai uzreiz pāriet uz izvēlni.
Iziet (exit) - var uzreiz iziet no bibliotēkas, un programma tiek pabeigta.
